    전체글 글쓴이: 심 형근 » 2002-09-12 21:21

    Height Map 을 작성하고 있는데, 한가지 심각한 문제는

    입력파일이 없다는 것입니다. 이미 존재하는 유명한 Height Map 파일을 로드해서, 외부지형을 렌더링 하려고 하는데.

    혹시, 지형(Height Map 으로 저장하는) 을 디자인하는 유명한 프로그램이 있나요? 그런 프로그램이 있으면, 만든 파일을 로드해서 사용하려고 합니다.

    2 가지 외부지형 렌더링 튜토리얼을 봤는데. 둘다 RAW 파일포맷을 사용하는군요. (RAW 파일포맷으로는 부족한게 많습니다.)

    지형 에디터를 만드는것도 상당한 시간을 소비하는 일인것 같고.

    혹시, 3DS 파일을 HeightMap 으로 바꾸는 방법이나.
    3D Studio Max 에서 HeightMap 으로 출력하는 방법도 좋습니다.

              혹시 MAX 사용하실 수 있나요?

              사용 가능 하시다면 MAX의 object중에 plane이란게 있습니다.

              여기에 Displace modifier를 넣어서 사용하시면 됩니다.

              주) Displace modifier: gray scale image에서 vertex의 높이를 정하는 modifier(displacemap이라고도 합니다)

              이렇게 하시면 원하는 지형을 간단히 만들 수 있습니다.

              또는 직접 vertex의 높이를 조정하시던지....ㅡㅡ;


              그럼 만들어진 지형을 파일로 만들어야 하는데......

              혹시 MAXScript 사용하실 수 있나요? ㅡㅡ;

              사용하실 수 있다면 아래의 MAXScript를 이용하면 각 지형의 높이 정보를

              파일로 만들 수 있습니다.


              -- (c)2001 KOG Inc.
              -- write Heightmap data file from Plane(Editable_Mesh)
              -- parameters: object
              -- nWid, nLen - # of width and height vertices respectively.
              -- written by ctkim@kogsoft.com

              fn xptHeightMap object nWid nLen = (

              --s = getSaveFileName caption:"Save HeightMap" filename:"heightmap.hm" \
              -- types:"HeightMap(*.hm)|*.hm"
              s = "d:\\programming\\terrainRendering\\" + object.name + ".hm"

              if s == undefined then (
              print "CANCELED: xptHeightMap()"
              return -1
              ) -- if s
              try (
              fs = createFile s

              -- write Header

              -- orign's position
              v = getVert object 1
              format "%\n" v.x to:fs
              format "%\n" v.y to:fs

              -- # of vertices
              format "%\n" nWid to:fs
              format "%\n" nLen to:fs

              -- terrain's size
              v = object.max - object.min
              format "%\n" v.x to:fs
              format "%\n" v.y to:fs

              nVerts = getNumVerts object
              for i = 1 to nVerts do (
              v = in coordSys object.transform getVert object i
              format "%\n" v.z to:fs
              ) -- for i

              flush fs
              close fs
              ) catch (
              print "ERROR : xptHeightMap"
              return -1
              )
              print "OK : xptHeightMap()"
              ) -- fn xptHeightMap()


              여기까지 입니다.

              저희 회사는 Level editor를 MAXScript로 만들어 사용하고 있습니다.
